What is HaldAI?

HaldAI is an "Obsidian-meets-Trello" productivity OS designed for deep work. It combines a drag-and-drop Kanban board, a rich-text notebook, and a context-aware AI agent into a single, beautiful interface.

Flow State

A clutter-free dark mode interface designed to keep you in the zone.

Second Brain

Capture ideas instantly in notebooks without leaving your task view.

AI Agent

An intelligent partner that plans your day and manages your board.

Quick Start

Get up and running in less than 60 seconds.

  1. Create a Board: Your workspace starts with a default "Operations" board.
  2. Add a Task: Click + ADD NODE to create your first task.
  3. Start a Timer: Click the edit icon on a task and set a duration (e.g., 25 mins) to begin a focus session.
  4. Ask AI: Open the Agent tab and type "Plan my day for coding" to auto-generate tasks.

Interface Tour

HaldAI is divided into three main areas:

  • The Canvas (Center): Your infinite Kanban board. Pan with middle-click or spacebar + drag. Zoom with scroll.
  • The Sidebar (Left): Quick access to Notebooks and different workspaces.
  • The Agent (Floating): Your AI companion, accessible via the "Agent" tab or Ctrl + /.

Task Cards

Tasks are the atomic units of work in HaldAI. They live in columns (Sections) and can be dragged around freely.

Features

  • Drag & Drop: Move tasks between columns to change status.
  • Timers: Tasks change visual state when active or overdue.
  • Subtasks: Break down complex tasks using the AI agent.

Notebooks

The Notebook is a distraction-free writing environment. It supports rich text and slash commands.

Type / in the editor to open the command menu. You can insert headings, to-do lists, quotes, and more.

/heading 1
/todo
/quote
